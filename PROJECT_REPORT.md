Project Report: Nassau Candy Advanced Logistics Dashboard
1. Project Overview
The Nassau Candy Advanced Logistics Dashboard is an end-to-end data analytics application designed to optimize supply chain operations. It allows stakeholders to monitor logistics performance, analyze sales trends, and identify efficiency bottlenecks using interactive visualizations.
2. Problem Statement
Managing complex logistics data involves handling massive volumes of information regarding shipping modes, delivery lead times, and regional profitability. Without a centralized analytical tool, identifying underperforming routes or products becomes a manual and error-prone task. This project addresses this by centralizing data insights into a user-friendly web interface.
3. Technical Stack
Language: Python
Web Framework: Streamlit
Data Manipulation: Pandas, NumPy
Visualization: Plotly, Streamlit Native Charts
Deployment: Streamlit Cloud
Version Control: Git & GitHub
4. Key Features
Executive Overview: Real-time metrics tracking Total Sales, Average Lead Time, and Total Profit.
Geographic Performance: Interactive heatmaps visualizing profit distribution across different states/provinces.
Advanced Filtering: Multi-select sidebar filters for State, Shipping Mode, and Date Range.
Operational Efficiency: Automated calculation of product-specific efficiency metrics.
5. Methodology
Data Ingestion & Cleaning: Imported the logistics dataset; handled missing values and converted date columns to datetime format for time-series compatibility.
Exploratory Data Analysis (EDA): Performed aggregation and grouping to identify key performance indicators (KPIs).
Visualization Design: Integrated Plotly to create responsive and interactive charts that allow users to drill down into specific data points.
Deployment: Linked the GitHub repository to Streamlit Cloud to enable continuous deployment and public access via a live URL.
