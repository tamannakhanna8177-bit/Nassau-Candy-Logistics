# 📖 Project Overview

The **Nassau Candy Logistics Dashboard** is a Business Analytics project developed using **Python** and **Streamlit**.

The dashboard transforms raw logistics data into meaningful insights that help businesses monitor shipment performance, warehouse efficiency, inventory trends, and delivery operations.

This project demonstrates practical skills in Data Analytics, Dashboard Development, and Business Intelligence.

- Analyze logistics operations
